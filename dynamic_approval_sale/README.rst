dynamic_approval_sale
======================

This module extends the sales workflow with a dynamic single-level approval process.

Features
--------
- Adds approval request before confirming a Sale Order.
- Supports primary and secondary approvers.
- Sends email and in-app notifications for approval.
- Restricts quotation actions until approved.
- Approval status visible on the Sale Order.

Usage
-----
1. Set up approval rules for Sale Orders.
2. When a quotation is created, an approval request is generated.
3. Approver receives a notification and can approve or reject the request.
4. Only approved orders can proceed to confirmation or quotation actions.
