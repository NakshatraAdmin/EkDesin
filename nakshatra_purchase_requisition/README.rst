=============================================
Module Name (Nakshatra Purchase Requisition)
=============================================

This module adds functionality to manage purchase requisitions.

Features
========

* Create PRs from Sale Orders
* Explode multi-level BoMs into requisition lines

Usage
=====

Go to Purchase → Orders → Purchase Requisition → New.
Go to Purchase → Orders → Purchase Requisition Items.
