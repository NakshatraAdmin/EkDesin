dynamic_approval_rule
=====================

This module provides a flexible dynamic approval system for any Odoo model.

Features
--------
- Supports single-level and multi-level approvals.
- Sequential or parallel approval flows.
- Primary and secondary approvers.
- Email and in-app notifications.
- Auto-approve, reject, or reassign on no response.

Usage
-----
1. Create approval rules with conditions and approvers.
2. Requests are automatically routed through the approval chain.
3. Approvers can approve or reject directly from the form view.
